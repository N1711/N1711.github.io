#Interactive Desk Bulder App to allow the user to choose a prefered design and color.
1. Click on Templates and select a desk style
2. Click on color and choose a desk texture
3. Click on the room icon at the bottom to see how it will look like in a room environment

This code can be used as a starting point if a programmer wants to develop a webpage with some customizable 3D objects content. For example a car dealer website which contains 3D models of the cars and still allow the user to change the color of the vehicle.
